## 🚀 Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

> Hot reload is enabled automatically via Vite.

## ✨ Features

- 🌍 150+ currencies supported
- 🔄 Swap From/To currencies instantly with amount sync
- 📡 Live exchange rates via [fawazahmed0 Currency API](https://github.com/fawazahmed0/exchange-api)
- 🔢 Decimal-safe input (type any amount including decimals)
- ⚡ Fast and lightweight (Vite)
